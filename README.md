# Project 2: Exploratory Data Analysis (EDA) Report
---
## Assigned Topic:
__[World Development Statistics]__

## Selected Problem Statement: 
Targeted Funding for "Hollow Growth" Regions
__This project analyzes historical population, GNI per capita, and life expectancy data (1960–2020) to identify countries experiencing "hollow growth." The findings support evidence-based recommendations for allocating the proposed $100 million intervention fund to countries with stalled human development.__
